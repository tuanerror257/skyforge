# Skyforge ✈️

Game không chiến 3D arcade chạy thẳng trên trình duyệt — một file HTML duy nhất, không build step, không asset ngoài.

- **Engine:** Three.js r160+ (CDN, ES modules)
- **Đồ họa:** 100% procedural low-poly, terrain 3 vùng (núi / hoang mạc / biển đảo)
- **Âm thanh:** Web Audio API
- **Chạy:** mở `index.html` bằng Chrome, hoặc chơi bản deploy trên Vercel

## Điều khiển
| Phím | Chức năng |
|---|---|
| W / S | Chúc / ngóc mũi |
| A / D | Nghiêng cánh (roll) |
| Q / E | Bẻ lái (yaw) |
| Shift / Ctrl | Tăng / giảm ga |
| Space | Bắn pháo |
| F | Bắn tên lửa |
| C | Đổi góc nhìn |
| R | Chơi lại |
